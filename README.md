# Healthcare KPI Tracker Dashboard

## Overview
Built an Excel-based healthcare operations dashboard to monitor hospital KPIs such as:
- Bed Occupancy Rate
- Average Length of Stay (ALOS)
- Patient Wait Time
- Outpatient Trends
- Department-wise Admissions

## Tools Used
- Microsoft Excel
- Pivot Tables
- Dashboard Design
- KPI Reporting
- Conditional Formatting

## Key Insights
- Department performance trends
- Monthly patient flow
- Admission type distribution
- Operational healthcare analytics

## Outcome
This project simulates hospital management reporting and demonstrates healthcare data analytics capabilities.
